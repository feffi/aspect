Filter `upper`
* no args

---

The `upper` filter converts a value to uppercase:

```twig
{{ 'welcome'|upper }}

{# outputs 'WELCOME' #}
```

**Note.** For UTF8 strings you have to install [luautf8](https://luarocks.org/modules/dannote/utf8) package
